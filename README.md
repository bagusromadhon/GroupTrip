# Tour Grip - Premium Travel Agency

Ini adalah repositori untuk tugas **Daily Project 7** (Berdasarkan rancangan Daily Project 6).
Website "Tour Grip" adalah prototipe *Landing Page* untuk agensi Travel & Tour eksklusif, dibangun menggunakan **HTML5, Vanilla CSS, dan Vanilla JavaScript** dengan mengedepankan estetika UI/UX premium (*Dark Mode*, *Glassmorphism*, dan *Scroll Animations*).

---

## 🧪 Tabel Pengujian Aplikasi

Pengujian dilakukan berdasarkan standar aspek kualitas perangkat lunak yang relevan untuk proyek pengembangan web.

| No | Aspek Pengujian (Quality Aspect) | Skenario Pengujian | Hasil yang Diharapkan | Status / Hasil Aktual |
|---|---|---|---|---|
| 1 | **Functional Suitability** | Mengklik tombol navigasi menu (Home, Destinations, Experiences) | Halaman *scroll* otomatis dengan mulus (*smooth scrolling*) ke bagian yang dituju. | ✅ **PASS** - Berfungsi dengan baik |
| 2 | **Functional Suitability** | Mengklik tombol "Hamburger" menu pada layar *mobile* | Menu navigasi *sidebar* terbuka dan animasi ikon berubah menjadi tanda silang (X). | ✅ **PASS** - Berfungsi dengan baik |
| 3 | **Usability** | Mengevaluasi estetika antarmuka dan kontras warna (Dark Mode) | Teks dapat terbaca dengan jelas, elemen visual tidak tumpang tindih, interaksi hover memberikan umpan balik (feedback) visual. | ✅ **PASS** - Kontras sangat baik, *glassmorphism* ter-render |
| 4 | **Performance Efficiency** | Memuat halaman website dari awal (*Hard Refresh*) | Gambar (*hero* dan *destinations*) serta font memuat dengan cepat tanpa *layout shift* yang mengganggu. Animasi *scroll* berjalan di 60 FPS tanpa *lag*. | ✅ **PASS** - Ringan dan mulus (Vanilla JS/CSS) |
| 5 | **Portability / Responsiveness** | Mengubah ukuran layar/jendela *browser* dari *Desktop* ke ukuran *Tablet* (768px) dan *Mobile* (320px - 480px) | *Layout grid* beradaptasi menjadi 1 kolom.
